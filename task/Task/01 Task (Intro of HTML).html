WEBSITE

A website is a collection of web pages and related content that is identified by a commom domain name and published on at least one web server.

WWW

WWW all the public websites or pages that users can access on their local computers and other devices through the internet.

HTTP/HTTPS

http (Hypertext transfer protocol) is a protocal using which hypertext is transferred over the web.http has been the most widely used protocol for data transfer over the web but the data exchanged using http is not as secure as we would like it to be. (Hypertext transfer protocal secure) is an extension of the Hypertext transfer protocal. its used for secure communication.In Https the communication protocol is encrypted using Transport Layer Security.

CLIENT SERVER

Client-server denotes a relationship between cooperating programs in an appliction, composed of clients intiating requests for services and servers providing that function or service.

DOMAIN

A DOMAIN is an easy to remember name that's associated with a physical IP address on the internet.

HOST

A host is a computer or other device that communicates with other hosts on a network.Also known as network hosts,hosts include clients and servers that send or receive data,services and applications.

HOSTING

Web hosting is a service that allows organizations and individuals to post a website or web page onto the Internet. A web host, or web hosting service provider, is a business that provides the technologies and services needed for the website or webpage to be viewed in the Internet.

FTP

File Transfer Protocol is a standard communication protocol used for the transfer of computer files from a server to a client on a computer network

PATH

A Path to access a file, or a lacation to store file.

EXTENSION

A file extension is the suffix of a computer file that an operating system uses to identify the exact file type.

EDITOR/VSCODE

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows

URL-URI/URN

A URI (Uniform Resource Identifier) is a string identifier that refers to the identity of a resource on the internet using either a location, name, or both. A URI further has two subsets, a URL (Uniform Resource Locator) and a URN (Uniform Resource Number).

ADDRESS BAR

The familiar text field at the top of a web browser's Graphical User Interface (GUI) that displays the name or the URL (Uniform Resource Locator) of the current web page.

TITTLE BAR

A title bar is a small strip that extends across the top of a window
